<h2><a id="user-content-features" class="anchor" href="#features" aria-hidden="true"><span class="octicon octicon-link"></span></a>Features</h2>

<ul>
<li>One-to-one mapping with REST API and the other official clients</li>
<li>Generalized, pluggable architecture. See <a href="http://www.elastic.co/guide/en/elasticsearch/client/javascript-api/current/extending_core_components.html">Extending Core Components</a></li>
<li>Configurable, automatic discovery of cluster nodes</li>
<li>Persistent, Keep-Alive connections</li>
<li>Load balancing (with pluggable selection strategy) across all available nodes.</li>
</ul>

<h2><a id="user-content-supported-elasticsearch-versions" class="anchor" href="#supported-elasticsearch-versions" aria-hidden="true"><span class="octicon octicon-link"></span></a>Supported Elasticsearch Versions</h2>

<p><a href="http://build-eu-00.elasticsearch.org/job/es-js_nightly/"><img src="https://camo.githubusercontent.com/461bed11d6c13f320005891c31e0933ef20dd1e0/687474703a2f2f696d672e736869656c64732e696f2f62616467652f656c61737469637365617263682d302e39302d2d322e302d677265656e2e7376673f7374796c653d666c61742d737175617265" alt="Supporting Elasticsearch Version 0.90 to 2.0" data-canonical-src="http://img.shields.io/badge/elasticsearch-0.90--2.0-green.svg?style=flat-square" style="max-width:100%;"></a></p>

<p>Elasticsearch.js provides support for, and is regularly tested against, Elasticsearch releases 0.90.12 and greater. We also test against the latest changes in several branches in the Elasticsearch repository. To tell the client which version of Elastisearch you are using, and therefore the API it should provide, set the <code>apiVersion</code> config param. <a href="http://www.elastic.co/guide/en/elasticsearch/client/javascript-api/current/configuration.html#config-options">More info</a></p>

<h2><a id="user-content-examples" class="anchor" href="#examples" aria-hidden="true"><span class="octicon octicon-link"></span></a>Examples</h2>

<p>Create a client instance</p>

<div class="highlight highlight-source-js"><pre><span class="pl-k">var</span> elasticsearch <span class="pl-k">=</span> <span class="pl-c1">require</span>(<span class="pl-s"><span class="pl-pds">'</span>elasticsearch<span class="pl-pds">'</span></span>);
<span class="pl-k">var</span> client <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-en">elasticsearch.Client</span>({
  host<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>localhost:9200<span class="pl-pds">'</span></span>,
  log<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>trace<span class="pl-pds">'</span></span>
});</pre></div>

<p>Send a HEAD request to <code>/?hello=elasticsearch</code> and allow up to 1 second for it to complete.</p>

<div class="highlight highlight-source-js"><pre><span class="pl-smi">client</span>.<span class="pl-en">ping</span>({
  <span class="pl-c">// ping usually has a 3000ms timeout</span>
  requestTimeout<span class="pl-k">:</span> <span class="pl-c1">Infinity</span>,

  <span class="pl-c">// undocumented params are appended to the query string</span>
  hello<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">"</span>elasticsearch!<span class="pl-pds">"</span></span>
}, <span class="pl-k">function</span> (<span class="pl-smi">error</span>) {
  <span class="pl-k">if</span> (error) {
    <span class="pl-en">console</span>.<span class="pl-en">trace</span>(<span class="pl-s"><span class="pl-pds">'</span>elasticsearch cluster is down!<span class="pl-pds">'</span></span>);
  } <span class="pl-k">else</span> {
    <span class="pl-en">console</span>.<span class="pl-c1">log</span>(<span class="pl-s"><span class="pl-pds">'</span>All is well<span class="pl-pds">'</span></span>);
  }
});</pre></div>

<p>Skip the callback to get a promise back</p>

<div class="highlight highlight-source-js"><pre><span class="pl-smi">client</span>.<span class="pl-c1">search</span>({
  q<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>pants<span class="pl-pds">'</span></span>
}).<span class="pl-en">then</span>(<span class="pl-k">function</span> (<span class="pl-smi">body</span>) {
  <span class="pl-k">var</span> hits <span class="pl-k">=</span> <span class="pl-smi">body</span>.<span class="pl-smi">hits</span>.<span class="pl-smi">hits</span>;
}, <span class="pl-k">function</span> (<span class="pl-smi">error</span>) {
  <span class="pl-en">console</span>.<span class="pl-en">trace</span>(<span class="pl-smi">error</span>.<span class="pl-smi">message</span>);
});</pre></div>

<p>Find tweets that have "elasticsearch" in their body field</p>

<div class="highlight highlight-source-js"><pre><span class="pl-smi">client</span>.<span class="pl-c1">search</span>({
  index<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>twitter<span class="pl-pds">'</span></span>,
  type<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>tweets<span class="pl-pds">'</span></span>,
  body<span class="pl-k">:</span> {
    query<span class="pl-k">:</span> {
      match<span class="pl-k">:</span> {
        body<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">'</span>elasticsearch<span class="pl-pds">'</span></span>
      }
    }
  }
}).<span class="pl-en">then</span>(<span class="pl-k">function</span> (<span class="pl-smi">resp</span>) {
    <span class="pl-k">var</span> hits <span class="pl-k">=</span> <span class="pl-smi">resp</span>.<span class="pl-smi">hits</span>.<span class="pl-smi">hits</span>;
}, <span class="pl-k">function</span> (<span class="pl-smi">err</span>) {
    <span class="pl-en">console</span>.<span class="pl-en">trace</span>(<span class="pl-smi">err</span>.<span class="pl-smi">message</span>);
});</pre></div>

<p>More examples and detailed information about each method are available <a href="http://www.elastic.co/guide/en/elasticsearch/client/javascript-api/current/index.html">here</a></p>
